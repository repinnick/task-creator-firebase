# Simple task creator
<hr>

<h4>You can create, complete and delete tasks.</h4>

`Firebase hosting:` [--click--](https://react-task-creator.web.app/current)

### Technologies
- `HTML5`
- `React`
- `Redux`
- `Firebase`
- `Git`
- `WebStorm`

### Installation
- `git clone <repository-url>`
- `cd task-creator-firebase`
- `npm install`

### Start
- `npm start` or `npm run start`
